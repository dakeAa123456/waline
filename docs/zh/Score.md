## **_响应示例_**

```ts
{
	"errCode": 0,
	"errMsg": "新增成功",
	"data": {
		"id": "6543c3a49755e3e35f6dd23f"
	},
	"timeCost": 133
}
```

| 序号 |  参数名  | 参数类型 |       描述说明        |
| :--: | :------: | :------: | :-------------------: |
|  1   | errCode  | Integer  |   0：成功 400：失败   |
|  2   |  errMsg  |  String  |     查询结果描述      |
|  3   | timeCost | Integer  |     接口消耗时间      |
|  4   |   data   |  Object  | 评分 ID（无实际意义） |
