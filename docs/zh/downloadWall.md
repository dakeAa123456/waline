## **_响应示例_**

```ts
{
	"errCode": 0,
	"errMsg": "下载成功",
	"data": {
		"id": "6549064d337a9fe98790d268"
	},
	"timeCost": 146
}
```

| 序号 |  参数名  | 参数类型 |         描述说明          |
| :--: | :------: | :------: | :-----------------------: |
|  1   | errCode  | Integer  |     0：成功 400：失败     |
|  2   |  errMsg  |  String  |       查询结果描述        |
|  3   | timeCost | Integer  |       接口消耗时间        |
|  4   |   data   |  Object  | 下载记录 ID（无实际意义） |
